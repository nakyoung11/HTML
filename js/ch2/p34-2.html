자바 스크립트를 작성하는 규칙

1. 대소문자를 구분해서 작성!
2. 들여쓰기를 해서 작성한다.
3. 행의 끝은 ;으로 마무리
4. 주석을 사용하여 설명을 작성
5. 식별자(identifier)는 정해진 규칙을 지켜 작성
ex) 변수, 함수, 속성, 클래스...
6. 예약어는 식별자로 사용할 수 없다.

변수

변수: 값(데이터)를 저장하는 공간

변수를 선언하는 규칙

1. 이름을 의미있게 짓는다.
2. 변수 이름은 Camel Case
3. 변수의 이름은 대소문자를 구분하다
4. 변수 이름의 첫 글자는 영어, _ , $를 사용.
$는 시스템 변수에서 자주 사용되기 때문에 가능하면 NO!
5. 예약어 사용 NONO
